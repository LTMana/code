# 功能

- 离线保存笔记
- 云端数据同步, 更换了设备也可以找到以前的笔记
- 接入了[好推二维码](https://weixin.hotapp.cn)提供的数据统计工具, 可以到平台上查看用户分析、留存分析、事件分析
